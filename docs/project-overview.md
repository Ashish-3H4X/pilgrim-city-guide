Project Name: Pilgrim City Guide

Problem:
Pilgrims visiting cities like Varanasi, Ayodhya, Mathura face problems such as:
- temple timings
- rules (mobile restrictions)
- free clockroom or Paid clockroom
- finding cheap stay
- navigation between temples
- nearby famous places

Solution:
A digital platform that provides complete pilgrimage guidance.

Core Features:
1. Temple information
2. Navigation and route guidance
3. Cheap stay and dharamshala info
4. Free night space 
5. Free food place
4. Food places
5. Nearby famous locations
6. Real time map navigation

Platforms:
- Web Application
- Android / iOS Mobile App

Tech Stack:
Frontend: React
Backend: Node.js + Express
Database: MongoDB
Mobile: React Native
Maps: Google Maps API
